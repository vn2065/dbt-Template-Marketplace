# Pricing Strategy

## Template Prices

| Template | Price | Rationale |
|----------|-------|-----------|
| SaaS Metrics | $149 | Most universal — largest addressable market |
| eCommerce Funnels | $129 | Slightly lower — more DIY tooling competition |
| Fintech Analytics | $179 | Higher complexity + compliance element = higher value |
| **Bundle (all 3)** | **$349** | ~22% discount, encourages higher cart value |

## Value Anchor

**Benchmark:** A senior analytics engineer costs $80-120/hr.
Building each template from scratch takes 40-80 hours.
That's $3,200–$9,600 in engineering time.

These templates start at $129 — a 96% discount on the equivalent build.

Use this framing in your copy.

## Discount Strategy

- **Launch week**: 20% off with code `LAUNCH20`
- **Newsletter subscribers**: 15% off exclusive code
- **Bundle discount**: Already built into $349 bundle price
- **Seasonal**: Black Friday 30% off

## Gumroad Setup

- Enable "Pay what you want" with $X minimum? **No** — anchors value clearly
- Offer refunds? **Yes, 14 days** — reduces purchase anxiety, rarely used for digital goods
- Affiliate commission? **30%** — good for influencer/newsletter distribution
- Limit purchases? **No** — unlimited
